LINUX/GNU

Коротка інформація про LINUX/GNU

LINUX/GNU – це вільна та відкрита операційна система, яка поєднує ядро Linux і набір програм GNU. Вона відома своєю стабільністю, безпекою та гнучкістю. Linux широко використовується на серверах, у розробці, наукових дослідженнях, а також у настільних і мобільних системах.

Перелік найбільш популярних дистрибутивів

  • Ubuntu: простий, універсальний, базується на Debian.
  • Debian: стабільний, основа для багатьох інших систем.
  • Fedora: сучасний, підтримується Red Hat.
  • Arch Linux: мінімалістичний, для досвідчених.
  • Manjaro: спрощений Arch для користувачів.
  • Linux Mint: зручний для новачків, схожий на Windows.
  • Kali Linux: спеціалізований для безпеки.

Приклади команд роботи в Linux CLI

pwd

показує поточну директорію

echo

виводить текст або змінну

ls

список файлів у директорії

cd

перехід між директоріями

touch

творення порожнього файлу

mkdir

створення нової папки

cp

копіювання файлів/папок

mv

переміщення або перейменування

rm

видалення файлу

history

показує історію введених команд
